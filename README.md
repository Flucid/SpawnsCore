# SpawnsCore
![alt text](https://raw.githubusercontent.com/Flucid/SpawnsCore/master/icon.png)
SpawnsCore, set a spawn in a variety of commands! Simple.

# Commands :
* Set a spawn

  * /sc sethub
  * /sc setspawn
  * /sc setlobby
* Reset a spawn

  * /sc resethub
  * /sc resetspawn
  * /sc resetlobby
