# SpawnsCore
SpawnsCore, set a spawn in a variety of commands! Simple.

# Commands :
* Set a spawn

  * /spawns sethub
  * /spawns setspawn
  * /spawns setlobby
* Reset a spawn

  * /spawns reset hub
  * /spawns reset spawn
  * /spawns reset lobby
